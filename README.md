I developed a web application, a ski resort management and booking system, which allows users to rent ski equipment and book available accommodations within the resort.
The application includes core features such as displaying all available ski equipment, filtering items based on relevant criteria, adding products to a shopping cart, and completing the rental process. Additionally, users can view and book resort accommodations, each presented with detailed information and a price per night.
The project was built using React and JavaScript, while the user interface was styled with Tailwind CSS and HTML. Data management and backend simulation were implemented using JSON Server, which handles all API requests.
